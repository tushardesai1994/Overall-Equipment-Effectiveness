# 🏭 Overall Equipment Effectiveness (OEE) – Power BI Dashboard
An end-to-end manufacturing performance analytics solution built in Power BI, focused on measuring and analyzing Overall Equipment Effectiveness (OEE) across machines, operators, time periods, and incidents.

This project demonstrates strong data modeling, DAX calculations, and manufacturing KPI analysis using a star schema approach.

🎯 Business Value

This dashboard enables:

  ▫️ Identification of production bottlenecks
  
  ▫️ Machine performance comparison
  
  ▫️ Operator efficiency tracking
  
  ▫️ Incident impact analysis
  
  ▫️ Shift-level performance evaluation
  
  ▫️ Data-driven production optimization

🛠 Tools Used

  ▫️ Power BI Desktop
  
  ▫️ DAX
  
  ▫️ Star Schema Modeling
  
  ▫️ Manufacturing KPI Framework (OEE)

📈 Key Metrics Implemented

  ▫️ Total Hours
  
  ▫️ Productive vs Outage Hours
  
  ▫️ Quantity Produced & Rejected
  
  ▫️ Planned vs Actual Output
  
  ▫️ OEE Over Time (Trend Analysis)
  
  ▫️ OEE by Machine
  
  ▫️ Availability by Month
  
  ▫️ Outage Analysis by Incident
  
  ▫️ Shift-Based Analysis using Time Dimension

🧠 Advanced DAX Techniques Used

  ▫️ CALCULATE
  
  ▫️ SUMX
  
  ▫️ FILTER
  
  ▫️ RELATED
  
  ▫️ DIVIDE
  
  ▫️ Context transition handling
  
  ▫️ Auxiliary KPI measures for progress bar visuals
